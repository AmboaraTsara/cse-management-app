# Application de Gestion d'Œuvre Sociale (CSE)

Application SaaS pour la gestion des demandes d'aide sociale d'un CSE.

## Fonctionnalités

- **Authentification** : Login avec JWT, rôles (ADMIN/MANAGER/BENEFICIARY)
- **Gestion des demandes** : Création, soumission, approbation/rejet, paiement
- **Budget annuel** : Configuration et vérification automatique
- **Audit log** : Traçabilité de toutes les actions

##  Stack Technique

- **Frontend** : React + TypeScript + Material-UI
- **Backend** : Node.js + Express + TypeScript
- **Base de données** : PostgreSQL
- **Authentification** : JWT + bcrypt
